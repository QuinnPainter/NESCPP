# NESCPP
NES emulator written in C++ using SFML for input, graphics and sound.
## Features
* Instruction-accurate CPU
* Cycle-accurate PPU
* Mostly working sound
* Supported mappers:
  * Mapper 0
  * Mapper 1
  * Mapper 2
